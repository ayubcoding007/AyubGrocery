AyubGrocery:-
AyubGrocery is a modern grocery e-commerce web application built using React.js. 
This project allows users to explore grocery products, browse categories, and add items to their cart easily. 
The interface is designed to be clean, simple, and user-friendly so customers can shop without confusion.

The application supports two roles: User and Seller. Users can register, log in, and purchase products, 
while the Seller has a separate login system (credentials stored securely in environment variables). 
This project demonstrates practical frontend skills including routing, API integration, 
authentication handling, cart management, and role-based UI rendering.

Technologies Used:-
Vite+React.js
Tailwind css
React Router DOM
Axios
Context API / State Management
JWT Authentication (Integrated with Backend)
